Claude Skill
Claude Skill

Qualitative Text Analysis

A structured qualitative analysis pipeline for Claude. Analyzes interview transcripts, focus groups, survey responses, and other unstructured text using rigorous methods — grounded theory coding, thematic analysis, and six-lens pattern detection. Produces traceable, auditable output instead of summaries.

  • Open coding with hierarchical tags (Domain / Category / Subcategory)
  • Theme construction with inclusion and exclusion boundaries
  • Six pattern lenses: similarity, difference, frequency, sequence, correspondence, causation
  • Cross-case participant matrix and typology detection
  • Full traceability — every quote verbatim, every code linked to source
  • Outputs 6 files: summary, themes, patterns, coding log, JSON data
